 悟空 (Wukong-AI) 🐒 —— 你的桌面情感陪伴大圣

这是一个基于树莓派硬件打造的实体 AI 助手，旨在通过语音与视觉交互，提供 24 小时的情感陪伴与生活关怀。

 🌟 项目灵魂
- 情感对话：接入豆包大模型，赋予悟空幽默、灵动、带点“猴气”的个性，能聊天、能解闷。
- 视觉陪伴：1.3寸 OLED 屏幕实时显示悟空的像素头像，提醒时更有专属形象展示。
- 硬核控制：自研按键控制流，支持短按唤醒、长按 2 秒紧急退出，交互随心所欲。
- 主动关怀：无需唤醒，到点自动播报提醒（起床、吃饭、睡觉），它是你的私人管家。

 🛠️ 技术实现
- 语音识别 (ASR): 使用 Whisper-cli 本地快速识别。
- 大脑 (LLM): 字节跳动 Doubao-Seed-1.6-Flash。
- 语音合成 (TTS): Edge-TTS 提供的自然男声（云希）。
- 音画同步: 针对 OLED 显存开发的同步分页算法。
- 多线程架构: 对话线程、按键监听线程、定时提醒线程互不干扰。

 📁 核心脚本说明
- `wukong_doubao_chat.py`: 对话主程序，负责听、想、说。
- `wukong_remind.py`: 独立值班脚本，负责到点主动喊人、显示悟空头像。
- `wukong_face.png`: 专门适配 OLED 的 128x64 悟空像素图。
- `test_face.py`: 快速预览图片显示效果的小工具。

 🚀 部署指南
1. 准备好树莓派及 SH1106 屏幕。
2. 上传 `wukong_face.png` 到 `/home/lamb/`。
3. 配置 `systemd` 服务，让两个脚本开机自启。
4. 剩下的，就交给这只猴子吧！
